# La Diversité - E-commerce de Bebidas 

## Sobre
Bem-vindo ao projeto La Diversité, um sistema de e-commerce para venda de bebidas. 
* **Funcionalidades:** Visualização de produtos, carrinho de compras, login, registro, finalização de compras com boleto e PIX.
* **Tecnologias:** React.js, Firebase, CSS, React Icons.

## Funcionalidades Principais
* **Visualização de Produtos:** Veja os detalhes dos produtos em modais, incluindo nome, imagem e descrição.
* **Carrinho de Compras:** Adicione produtos ao carrinho e visualize um resumo com preços e quantidades.
* **Login e Registro:** Crie uma conta com nome, CPF e senha ou faça login para finalizar suas compras.
* **Finalização de Compra:** Escolha métodos de pagamento (boleto ou PIX), calcule o total e conclua sua compra.
* **Persistência de Dados:** Dados dos usuários e pedidos armazenados no Firebase.

## Componentes Principais
* **DetalhesProdutoModal.js:** Modal que exibe os detalhes de um produto.
* **FinalizarCompraModal.js:** Modal para visualizar o carrinho e finalizar a compra.
* **LoginForm.js:** Formulário de login.
* **RegisterForm.js:** Formulário de registro de novos usuários.
  
## Funcionalidades Detalhadas
* **Autenticação:** Os usuários podem se registrar e logar utilizando CPF e senha. A verificação de CPF existente é feita antes de concluir o registro.

* **Finalização de Compra:** O modal de finalização de compra permite selecionar a forma de pagamento e visualizar o resumo dos itens no carrinho. Após a compra, os dados são salvos no Firebase.

* **Cadastro de Usuário**: O formulário de cadastro valida se o CPF já está registrado e se a senha e a confirmação de senha correspondem.
